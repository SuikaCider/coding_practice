# Jiki overview 

[Jiki](https://jiki.io/) is a precursor to [Exercism](http://exercism.org/), intended to help users understand key computer science concepts so that they can work with LLMs more effectively. It does not exactly provide users with the skills necessary to do any programming, but does give you some basic ability to look at a program or script and grok how it works.

In particular, it introduces the following concepts via a Lesson → Exercise format:
- Functions
- Variables
- Strings
- Colors 
- Loops
- Arrays 
- Dictionaries
- Animation 
- Random numbers
- Scope 
- Scenarios
- Methods
- Properties
